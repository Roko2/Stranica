HTML

HTML je osnovni jezik koji se upotrebljava za izradu web stranica.
To je jezik za označavanje hipertekstualnih dokumenata.
Služi za definiranje strukture tekstova, medija i ugrađenih objekata
u okviru web stranice i elektronske pošte.
Tekst i mediji se strukturiraju pomoću elemenata poput h1-h5, div,
p, img, link...
 Element h označava naslov i njegovu veličinu po brojevima
(1 je najveći).
Element div označava blok koji može biti bilo kojeg sadržaja itd.
Sve dodatne informacije nalaze se na stranici w3schools koja
opširno opisuje svaki postojani element, kako u HTML-u,
tako i u ostalim jezicima koji se koriste za izradu web stranice.

Stranica se sastoji od dijelova po kojima se slažu elementi, od vrha do dna.
Poznavanje tog rasporeda pomaže nam u postavljanju teksta, slika itd.
Vrh je head gdje se piše ime stranice koje se nalazi u navigacijskom oknu
preglednika, zatim se umeće znakovlje koje će preglednik obuhvaćat
i na kraju linkovi na vanjske dokumente koje želimo povezati sa HTML-om.
U body su svi elementi koji čine izgled stranice i sve se nalazi u HTML
elementu jer on govori da se radi o HTML dokumentu.
Kako bi elementi i sama stranica izgledali što ljepše vrlo bitan je
u svemu tome CSS dokument.
